Privacy Policy for WiseFido BLE Config Tool

Last Updated: 2025-04-03

1. Data Collection

This app only uses Bluetooth Low Energy (BLE) to:

  • Configure your WiseFido device's WiFi settings
  • Send server connection parameters

We do NOT:

  • Collect any personal information
  • Store WiFi passwords or server credentials
  • Track device location

2. Permissions

The app requires Bluetooth access solely to communicate with your WiseFido hardware.

3. Contact

For questions: support@wisefido.com
